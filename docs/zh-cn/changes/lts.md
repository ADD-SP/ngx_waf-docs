## [6.1.0] - 2021-08-03 UTC+0800

### 新增

* 为配置项 `waf_mode` 增加了三个选项。
    * ADV：控制开关高级规则。
    * CMN-METH：等价于 `HEAD GET POST`。
    * ALL-METH：任意的 HTTP 请求方法都会启动检查。
